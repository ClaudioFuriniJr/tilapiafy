# tilapiafy

![npm](https://img.shields.io/npm/v/tilapiafy.svg)

Generates a tilapia.

## Install

```
$ npm install tilapiafy
```

## Usage

```js
import React from 'react';
import Tilapia from 'tilapiafy';

class App extends React.Component {
  render() {
    return (
      <div className="App">
          <Tilapia />
      </div>
    );
  }
}

export default App;
```
